ToDo List:ToDo List - C# | MySQL | Dapper | JetBrains Rider

Feeling scattered, lost, distracted? Engage with my ToDo List, an organizational console app I created to practice not only the use of classes and interfaces, database integration, and data manipulation, but also refactoring my code towards better organization and readability.
Project Overview:
  1. Created the project using Rider
  2. Created a database in MySQL Workbench
  3. Utilized Dapper to integrate and manipulate data in the database
  4. Allows users to personalize their ToDo List, tracking progression and even scheduling.
