My FitnessApp - C# | HTML | CSS | JavaScript | SQL | Dapper | Entity Framework | MySQL Workbench | ASP.NET Core MVC | Test-Driven Development

This app is a work in progress aimed for increasing personal health! Currently, it holds a database of exercises intended as an exercise routine building starting point, with user ability to change each exercise to fit their goal and add new exercises as inspiration hits them. It also includes a database of user measurements. Optional fields allow users to customize their user experience.
Project Overview:
  1. Test-driven development utilized for the conversion ability.
  2. Created originally utilizing Dapper, I changed it to utilize Entity Framework instead so that in the future I can add a solid Login. I have not deleted the Dapper pages to demonstrate how they were written.
  3. Integrated several databases from MySQL workbench in order to achieve multiple uses for the app.
  4. Solid foundation on which to build various app abilities.
